2. Keyboard-Shortcuts für schnelleres Arbeiten im Debugger

Wichtige Shortcuts (Windows/Linux):

  • Step Over: F8
  • Step Into: F7
  • Step Out: Shift + F8
  • Run to Cursor: Alt + F9
  • Resume Program: F9
  • Toggle Breakpoint: Ctrl + F8
  • View Breakpoints: Ctrl + Shift + F8
  • Evaluate Expression: Alt + F8
  • Stop Debugging: Ctrl + F2
  • Switch Threads/Frames: Im Debug-Fenster per Maus oder Pfeiltasten

Wichtige Shortcuts (Mac):

  • Step Over: F8
  • Step Into: F7
  • Step Out: Shift + F8
  • Run to Cursor: Alt + F9
  • Resume Program: Command + Option + R
  • Toggle Breakpoint: Command + F8
  • View Breakpoints: Command + Shift + F8
  • Evaluate Expression: Option + F8
  • Stop Debugging: Command + F2
