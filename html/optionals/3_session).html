Handout: Optional erzeugen und verwenden

1. Optional mit Wert: Optional.of()

  • Erzeugt ein Optional, das garantiert einen Wert enthält.
  • Wenn der Wert null ist, wird sofort eine NullPointerException geworfen!
  • Verwende Optional.of() nur, wenn du sicher bist, dass der Wert nicht null ist.

Beispiel:

Optional<String> name = Optional.of("Anna");
System.out.println(name.isPresent()); // true

Optional<String> fehler = Optional.of(null); // -> NullPointerException!

2. Optional leer: Optional.empty()

  • Erzeugt ein leeres Optional, also ein Optional ohne Wert.
  • Praktisch, um eine "Kein Wert vorhanden"-Situation explizit darzustellen.

Beispiel:

Optional<String> leer = Optional.empty();
System.out.println(leer.isPresent()); // false

Typischer Anwendungsfall: Wenn eine Methode nichts finden konnte (z.B. Suche erfolglos).

3. Optional mit evtl. Nullwert: Optional.ofNullable()

  • Erzeugt ein Optional, das den Wert enthält, wenn er nicht null ist – andernfalls wird automatisch ein leeres Optional zurückgegeben.
  • Das ist die flexibelste Methode, um mit möglichen Nullwerten umzugehen.

Beispiel:

String wert = findeEtwas(); // könnte null sein!
Optional<String> eventuellWert = Optional.ofNullable(wert);

if (eventuellWert.isPresent()) {
    System.out.println(eventuellWert.get());
} else {
    System.out.println("Kein Wert vorhanden!");
}

Vergleich:

Optional<String> name1 = Optional.of("Bernd");        // Wert garantiert
Optional<String> name2 = Optional.empty();            // garantiert leer
Optional<String> name3 = Optional.ofNullable(null);   // auch leer
Optional<String> name4 = Optional.ofNullable("Clara");// Wert vorhanden

4. Praxis-Tipps

  • Nie mit Optional.of(null) arbeiten! Das führt sofort zu einer Exception.
  • Für Rückgabewerte aus Methoden, die etwas oder nichts liefern können, ist Optional.ofNullable() die beste Wahl.
  • Erzeuge keine Optional-Werte, wenn du stattdessen mit leeren Optional arbeiten kannst (Optional.empty()).

5. Übungen zum Selbststudium

  1. Erzeuge ein Optional aus einem garantiert nicht-null Wert und prüfe mit .isPresent(), ob ein Wert vorhanden ist.
  2. Schreibe eine Methode, die null zurückgeben könnte, und wandle sie so um, dass sie stattdessen ein Optional nutzt.
  3. Was passiert, wenn du Optional.of(null) aufrufst? Teste es und lies die Fehlermeldung.

6. Merksätze

  • Optional.of() – nur bei garantiert nicht-null Wert!
  • Optional.empty() – leeres Optional, kein Wert.
  • Optional.ofNullable() – nimmt Wert oder gibt bei null ein leeres Optional zurück.
