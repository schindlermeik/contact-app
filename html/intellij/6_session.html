FAQ: Was tun, wenn…? (Typische Anfängerprobleme in IntelliJ IDEA)

…mein Code plötzlich „rot“ ist und alles voller Fehler?

  • Schritt 1: Prüfe, ob du das richtige JDK ausgewählt hast (File → Project Structure → Project SDK).
  • Schritt 2: Versuche „File → Invalidate Caches / Restart…“
  • Schritt 3: Achte auf fehlende oder fehlerhafte Importe (Alt + Enter nutzen!)

…mein Projekt nicht baut oder Abhängigkeiten fehlen?

  • Schritt 1: Klicke im Maven- oder Gradle-Toolfenster auf „Reload Project“.
  • Schritt 2: Prüfe auf Tippfehler in der pom.xml oder build.gradle.
  • Schritt 3: Starte das Build über das Terminal mit mvn clean install oder gradle build.

…Shortcuts funktionieren nicht wie beschrieben?

  • Prüfe, ob dein System eigene Tastenkombinationen verwendet, die mit IntelliJ kollidieren (z. B. Ubuntu, macOS).
  • Schaue unter File → Settings → Keymap nach deiner aktiven Tastenbelegung.
  • Ändere Shortcuts, falls sie mehrfach belegt sind.

…ich nicht mehr weiß, wie eine Funktion heißt?

  • Strg + Shift + A / ⌘ + Shift + A drücken und eine Beschreibung oder einen Namen eingeben.

…die IDE plötzlich sehr langsam ist?

  • Unnötige Plugins deaktivieren.
  • RAM-Zuweisung erhöhen (über „Help → Change Memory Settings“).
  • Projekt- oder System-„Caches“ leeren: „File → Invalidate Caches / Restart…“

…meine Git-Aktionen nicht angezeigt werden?

  • Prüfe, ob Git im Projekt aktiviert ist (VCS → Enable Version Control Integration).
  • Überprüfe, ob du ein Repository initialisiert hast.
  • Aktualisiere das Toolfenster „Version Control“ und prüfe die Einstellungen unter File → Settings → Version Control → Git.

…ich Hilfe zu einem Symbol/Fehlermeldung brauche?

  • Strg + Q / F1 für Quick Documentation.
  • Rechtsklick → Help → Online Documentation.
  • Nutze die JetBrains-Hilfe: https://www.jetbrains.com/help/idea/

Tipp: Fast jedes Problem in IntelliJ lässt sich mit ein paar Minuten Suche in den Einstellungen oder mit den richtigen Shortcuts lösen – nutze die „Find Action“-Funktion und das Hilfemenü regelmäßig!
