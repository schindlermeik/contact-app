Handout: Praktische Übungen und Beispiele zur Streams API

1. Filter-, Map- und Reduce-Operationen auf Listen von Zahlen und Strings

a) Filtern: Nur gerade Zahlen herausfiltern

List<Integer> zahlen = Arrays.asList(1, 2, 3, 4, 5, 6);

List<Integer> gerade = zahlen.stream()
    .filter(x -> x % 2 == 0)
    .collect(Collectors.toList());

System.out.println(gerade); // [2, 4, 6]

b) Map: Alle Strings in Großbuchstaben umwandeln

List<String> namen = Arrays.asList("Anna", "Bernd", "Clara");

List<String> upper = namen.stream()
    .map(String::toUpperCase)
    .collect(Collectors.toList());

System.out.println(upper); // [ANNA, BERND, CLARA]

c) Reduce: Summe aller Zahlen berechnen

int summe = zahlen.stream()
    .reduce(0, (a, b) -> a + b);

System.out.println(summe); // 21

2. Komplexe Datentransformationen mit Streams

Beispiel: Aus einer Liste von Personen die Namen aller Erwachsenen (ab 18) in Kleinbuchstaben extrahieren

public class Person {
    private String name;
    private int alter;
    // Konstruktor, Getter, Setter...
}

List<Person> personen = Arrays.asList(
    new Person("Anna", 20),
    new Person("Bernd", 17),
    new Person("Clara", 22)
);

List<String> erwachseneNamen = personen.stream()
    .filter(p -> p.getAlter() >= 18)
    .map(p -> p.getName().toLowerCase())
    .collect(Collectors.toList());

System.out.println(erwachseneNamen); // [anna, clara]

3. Vergleich: Lösung mit Schleife vs. Lösung mit Streams

Aufgabe: Aus einer Liste alle Zahlen > 3 quadrieren und aufsummieren.

a) Mit Schleife

List<Integer> zahlen = Arrays.asList(1, 2, 3, 4, 5);
int summe = 0;
for (int x : zahlen) {
    if (x > 3) {
        summe += x * x;
    }
}
System.out.println(summe); // 41

b) Mit Stream

int summe = zahlen.stream()
    .filter(x -> x > 3)
    .map(x -> x * x)
    .reduce(0, Integer::sum);

System.out.println(summe); // 41

4. Anwendung von Gruppierung und Partitionierung

a) Gruppieren: Personen nach Alter gruppieren

Map<Integer, List<Person>> nachAlter = personen.stream()
    .collect(Collectors.groupingBy(Person::getAlter));

b) Partitionieren: In erwachsen und minderjährig aufteilen

Map<Boolean, List<Person>> partition = personen.stream()
    .collect(Collectors.partitioningBy(p -> p.getAlter() >= 18));

c) Durchschnittsalter berechnen

double durchschnitt = personen.stream()
    .collect(Collectors.averagingInt(Person::getAlter));
System.out.println(durchschnitt);

5. Übungen zum Selbststudium

  1. Finde mit Streams alle Strings aus einer Liste, die länger als 4 Zeichen sind, und gib sie sortiert aus.
  2. Multipliziere alle ungeraden Zahlen einer Liste miteinander (Tipp: reduce() mit Startwert 1).
  3. Gruppiere eine Liste von Personen nach dem Anfangsbuchstaben des Namens.
  4. Partitioniere eine Liste von Zahlen in gerade und ungerade Zahlen.
  5. Berechne mit Streams das längste Wort in einer Liste von Strings.

6. Merksätze

  • Streams sind ideal für komplexe Datenverarbeitung in wenig Codezeilen.
  • Viele typische Aufgaben (Filtern, Transformieren, Aggregieren, Gruppieren) werden so viel übersichtlicher und leichter wartbar.
  • Vergleiche deine Lösungen regelmäßig mit klassischen Schleifen – so erkennst du Vorteile und Grenzen.
