Handout: Erzeugung von Streams

1. Streams aus Collections erzeugen

Java Collections (wie List, Set) können ganz einfach in Streams umgewandelt werden. Die gängigsten Methoden sind:

a) .stream()

Erzeugt einen sequentiellen Stream aus der Collection. Beispiel:

List<String> namen = Arrays.asList("Anna", "Bernd", "Clara");
Stream<String> stream = namen.stream();

Nun kannst du auf diesem Stream weitere Operationen ausführen.

b) .parallelStream()

Erzeugt einen parallelen Stream, der mehrere Prozessorkerne nutzt – nützlich bei sehr großen Datenmengen. Beispiel:

Stream<String> parallelerStream = namen.parallelStream();

Hinweis: Nicht jeder Code profitiert von Parallelität – sie kann sogar langsamer sein, wenn die Aufgabe zu klein oder nicht thread-sicher ist.

2. Streams aus Arrays

Auch Arrays können bequem in Streams umgewandelt werden – mit Hilfe der Hilfsklasse Arrays.

Beispiel:

String[] farben = {"Rot", "Grün", "Blau"};
Stream<String> farbenStream = Arrays.stream(farben);

Für primitive Datentypen gibt es spezielle Streams:

int[] zahlen = {1, 2, 3};
IntStream zahlenStream = Arrays.stream(zahlen);

3. Streams mit Fabrikmethoden

Die Klasse Stream bietet mehrere statische Methoden, mit denen du Streams flexibel erzeugen kannst:

a) Stream.of()

Erzeugt einen Stream aus den übergebenen Argumenten.

Stream<String> stream = Stream.of("Apfel", "Banane", "Kirsche");

b) Stream.generate()

Erzeugt einen unendlichen Stream, bei dem jeder Wert durch einen Supplier erzeugt wird.

Stream<Double> zufallszahlen = Stream.generate(Math::random);
// Achtung: Unendlicher Stream – daher limit() nutzen!
zufallszahlen.limit(5).forEach(System.out::println);

c) Stream.iterate()

Erzeugt einen unendlichen (oder ab Java 9: endlichen) Stream, indem ein Startwert und eine Erzeugungsregel angegeben werden.

Stream<Integer> zahlen = Stream.iterate(0, n -> n + 2);
zahlen.limit(5).forEach(System.out::println); // 0, 2, 4, 6, 8

Ab Java 9 kann Stream.iterate() optional eine Abbruchbedingung bekommen:

Stream<Integer> zahlenBis10 = Stream.iterate(0, n -> n < 10, n -> n + 2);
zahlenBis10.forEach(System.out::println); // 0, 2, 4, 6, 8

4. Übungen zum Selbststudium

  1. Erzeuge einen Stream aus einer Liste von Namen und gib alle Namen in Großbuchstaben aus.
  2. Wandle ein Array von Ganzzahlen in einen Stream um und berechne die Summe aller Zahlen.
  3. Nutze Stream.of(), um einen Stream aus vier Farben zu erzeugen, und zähle die Elemente.
  4. Erzeuge mit Stream.generate() fünf Zufallszahlen zwischen 0 und 1 und gib sie aus.
  5. Erzeuge mit Stream.iterate() die ersten zehn geraden Zahlen und gib sie aus.

5. Merksätze

  • Collections und Arrays können leicht in Streams verwandelt werden.
  • Mit Fabrikmethoden sind auch Streams aus beliebigen Werten oder Generatoren möglich.
  • Unendliche Streams immer mit limit() begrenzen, sonst läuft das Programm endlos!
